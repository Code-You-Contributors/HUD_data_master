 # Meals Served in Louisville
 I am tracking the trend in how many meals are served in the Lousivlle area to people in need.

 # Project Overview
 There is a high demand in Louisville for people who do not eat on a regular basis, including students attending school.  I am analyzing how many meals are served, what zip codes give out the most meals, and what is the trend in serving meals.
 Using Tableau, I would like to also track the snow plow routes to help determine how accessible meals are during inclement weather.

 # Project Setup Instructions
 1. Clone the repository.
 2. Save the Folder.
 3. Run pip install -r requirements.txt
 4. Open Jupyter Notebook from the command line or start menu.
 5. Go to the where you saved the notebook.
 6. Open 'meals_served_hud.ipynb'
 7. Click Run All.
